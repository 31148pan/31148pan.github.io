# css属性的继承性

# font 系列

font-family、font-size、font-weight、font-style等。

# text 系列

text-indent、text-align、 line-height、 letter-spacing（字符间间距）、word-spacing（单词间距）、text-transform（文本大小写）、direction（文本方向）、color。

# 列表属性

list-style等。

# 元素可见性

visibility、opacity。

# 光标属性

cursor。

# 不能继承的属性

display、vertical-align、text-decoration、text-shadow、盒子模型的属性、背景属性、定位属性等。

**a 标签的字体颜色不能被继承**。
