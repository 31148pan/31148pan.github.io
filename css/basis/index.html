# CSS

# CSS定义及特性

    CSS (Cascading Style Sheets 缩写) 层叠样式表，what？那么CSS到底是用来做什么的呢？

更简单，它的诞生就是用来格式化和排版网页，负责装饰我们浏览的页面，让她看起来变得更美观一些。

# 三大特性

CSS 概括起来总共有三大特性。分别是：继承性、优先级和重叠性。

# 继承性

    所谓继承性就是给父元素设置样式后，它的所有子类元素会继承父类的样式。

  <div>
    <p>我是一段文字</p>
  </div>

    color: red

# 层叠性

    所谓层叠性就是在多个选择器选中"同一个标签", 然后又设置了"相同的属性", 就会发生层叠性。"后者覆盖前者",而层叠性由优先级决定。

  <div class="box1">
    <div class="box2">我是一段文字</div>
  </div>

# 优先级

    最后：优先级由权重决定。小括号内是css各种选择器的权重值。权重优先级默认从大到小排列顺序为：！important>行内样式（权重值为1，0，0，0）>Id（权重值为0，1，0，0）>类、伪类、属性选择器（权重值为0，0，1，0）>标签、伪元素选择器（权重值为0，0，0，1）>通用选择、子选择器 、相邻兄弟选择器、通用兄弟选择器（权重值为0，0，0，0）>继承>浏览器默认。

权重比较规则：     先从高等级进行比较，高等级相同时，再比较低等级的，以此类推；完全相同的话，就采用 后者优先 原则。例如：
body#god div.dad span.son {width: 200px;}
body#god span#test {width: 250px;}

    权重比较过程应该是在第二等级id选择器的比较就结束了：(#god + #test = 0,2,0,0) > (#god = 0,1,0,0)。所以比较权重时，就是先比较id选择器个数，如果id一样多，再比较class选择器个数。 在css中，!important的权重最高。如果出现了!important，则不管权重如何都取有!important的属性值。但是宽高有例外情况，由于宽高会被max-width/min-width覆盖，所以!important会失效。 例如： width: 100px!important; min-width: 200px;

会被引擎解析成：

width: 200px;

# CSS的使用方法

CSS 的使用方法总共有三种：内联样式、行内样式和外联样式。

# 内联样式

在 head 标签中的 titile标签下面嵌入style标签即可。

<style>
  ...
</style>
1
2
3

# 行内样式

通过设置目标元素的style属性（属性名：属性值，属性与属性之间用分号;隔开）。

<div style="key1:value2;key2:value2...">
  ...
</div>
1
2
3

# 外联样式

在一个css文件中写样式，将<link rel="stylesheet" href="外联样式的路径">标签插入head 标签中即可。

# CSS 注释

为了代码的可读性和可维护性。往往我们定义CSS属性时，会需要写一些注释，以便后来查看维护。

规则：使用 /* 注释内容 */ 即可。

vscode 快捷键：单行注释 ctrl + / 、多行注释 shift + alt +a
