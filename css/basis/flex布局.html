# flex布局

flexBox 弹性布局。（是一种用于适应不同屏幕大小以及设备类型的响应式网页布局方式），任何一种容易都可以指定为flex布局。但需要注意的是：开启弹性布局后，子元素的float、clear、vertical-align属性都会失效。

开启弹性布局：display：flex 或者 inline-flex。设置元素为弹性布局方式后，该元素为布局容器，其子元素（不包括孙子元素）为容器项目。默认情况子项目会沿主轴从左至右依次排列，子项目的宽高由内容决定。

# 容器属性

# flex-direction

属性用来定义容器主轴的方向，即项目的排列方向。值可为：

  • row 主轴为水平方向，起点为左端，项目会依次从左到右排列，默认值。

  • row-reverse 与row相反，子元素会从右向左排列，即最后一个排在最左边。

  • column 主轴方向为垂直方向（项目由上至下依次排列）。

  • column-reverse 与column相反，最后一个排在最上面。

# flex-wrap

属性用于指定容器内的项目的换行方式。

  • nowrap 默认，项目排在一条轴线上，不进行换行。

  • wrap 设置弹性容器为多行。该情况下项目溢出的部分会被放置到新行。第一行在上方。

  • wrap-reverse 与wrap类似，只是第一行在下方。

  • flex-flow flex-direction与flex-wrap的合写属性。默认值 flex-flow:row nowrap。

# justify-content

定义项目在主轴（横轴）方向上的对齐方式。

  • flex-start：弹性项目左对齐。

  • flex-end：弹性项目右对齐。

  • center：弹性项目居中紧挨着填充。

  • space-between：弹性项目两端对齐，项目之间的间距相等。如果剩余空间为负或者只有一个弹性项，则该值等同于flex-start。

  • space-around：项目两端间距相等且项目之间的间距是两端间距的2倍。如果剩余空间为负或者只有一个弹性项，则该值等同于center。

# align-items

设置项目在侧轴（交叉轴）方向上的对齐方式。

  • flex-start：与交叉轴起点对齐。

  • flex-end：与交叉轴终点对齐。

  • center：与交叉轴中心对齐。

  • stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度。

# align-content

属性用于设置多条轴线的对齐方式。如果项目只有一根轴线，该属性不起作用。

  • flex-start 各行向交叉轴起始位置堆叠。

  • flex-end 各行向交叉轴结束位置堆叠。

  • center 各行向交叉轴中间位置堆叠。

  • space-between 各行在交叉轴中平均分布。

  • space-around 轴线两端间距相等且轴线之间的间距是两端轴线间距的2倍。

  • stretch（默认值）：轴线占满整个交叉轴。

# 项目属性

# align-self

属性用于设置弹性元素自身在侧轴（纵轴）方向上的对齐方式。允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。

  • auto，默认值，表示继承父元素align-items属性。

  • flex-start：弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴起始边界。

  • flex-end：弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴结束边界。

  • center：弹性盒子元素在该行的侧轴（纵轴）上居中放置。（如果该行的尺寸小于弹性盒子元素的尺寸，则会向两个方向溢出相同的长度）。

# order

控制子项目的排列顺序，数值小的排在前面。可以为负值。默认为0。

# flex-grow

定义项目的放大比列，默认为0，即如果存在剩余空间也不放大。如果所有项目的该属性值都为1，它们将等分剩余空间。如果一个项目的该属性值为2，其他项目的该属性值为1，则前者所占据的剩余空间是其它项目的2倍。

# flex-shrink

定义项目的缩小比列，默认为1，即如果空间不足，该项目会缩小。如果所有项目的该属性值都为1，则空间不足时，它们会等比列缩小。如果一个项目的该属性值为0，其它项目的该属性值为1，则空间不足时，前者不缩小。

# flex-basis

属性定义了在分配多余空间之前，项目占据的主轴空间（main size）。浏览器根据这个属性，计算主轴是否有多余空间。它的默认值为auto，即项目的本来大小（宽高由内容决定）。

# flex

flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选。比如所有弹性子元素的数值总和为4，某一弹性子元素flex设置为2，即该弹性子元素占比是1/2。
