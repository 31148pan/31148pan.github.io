# 块级元素

水平居中一个或者多个块级元素。

# 居中的盒子定宽

基本代码（为了方便这里没有将css与结构分离）：

<div style="width:200px;height:100px;background-color:orange;">
  <div style="width:100px;height:50px;background-color:pink;">
  </div>
</div>

问题：如何将橘色盒子内的粉色盒子水平方向上居中？ 效果图如下：

  • 利用 居中元素 的左右外边距 margin-left 和 margin-right。将其设置为auto。

这里只需要在所需要居中的块级元素上添加一行属性：margin:0 auto，这里仅需在基本代码中的style属性添加该行属性即可水平居中。

  • 利用 position + 负 margin / translateX() 。设置“子绝父相”，令子元素向右移动父元素宽度的50%，在设置 margin 为 子元素宽度 的负一半 / 向左移动本身宽度的负一半。

这里设置第一个div元素定位属性为：position:relative 和第二个div元素的定位属性为： position:absolute;left:50%;margin-left:-50px 。

  • 利用 position + 左右外边距 auto。设置“子绝父相”，且绝对定位元素 的 left、right 都设置为 0，再设置 margin-left 和 margin-right 为 auto。

这里设置第一个div元素定位属性为：position:relative 和第二个div元素的定位属性为： position:absolute;left:0;right:0;margin-left:auto;margin-right:auto 。

  • 改变元素的显示方式。改变子元素显示方式。

    • 一、设置子元素 display 为 inline 或者 inline-block，再设置父元素 text-align:center。适用于多个块级元素居中;

    • 二、设置父元素 display 属性为 flex或inline-flex 、justify-content:center。

    • 三、设置父元素 display 属性为 flex或inline-flex 、margin: 0 auto。

# 居中的盒子不定宽

  • 利用 position + translate。设置“子绝父相”，先令子元素向右移动 父元素（父元素没设置宽度，一级一级向上继承） 宽度的50%（left:50%），在向左移动 子元素 宽度的负50%( transform:translate(-50%))。

  • 改变居中元素的显示方式。

    • 一、设置居中元素的 display 属性为 inline 或者 inline-block（此时居中元素显示方式类似行内元素）。所以，再对其父元素设置 text-align 属性为center。即可居中。适用于多个块级元素居中；

    • 二、设置父元素 display 属性为 flex或inline-flex 、justify-content:center。

    • 三、设置父元素 display 属性为 flex或inline-flex 、margin: 0 auto。

    • 四、设置居中元素的 display 属性为 table（此时元素显示方式类似 块级元素），所以再设置 margin-left:auto 和 margin-right:auto。

# 行内元素（多行内元素）

利用 text-align: center 可以实现文本水平居中。

# 垂直居中

# 行内元素

  • 单行内联元素垂直居中。直接设置 line-height 等于 height 即可。

  • 多行内联元素垂直居中。

    • 一、利用 flex 实现垂直居中，其中flex-direction: column定义主轴方向为纵向。再设置 justify-content：center 即可；

    • 二、利用 table。设置父元素的 display 为 table 、子元素的 display 为 table-cell 及 vertical-align 为 middle 即可。

# 块级元素

# 居中的盒子定高

  • 使用定位+负margin / translateY()。通过“子绝父相”，设置子元素距离顶部50%，并设置 margin-top 向上偏移元素高度的一半，即可。

  • 改变元素的显示方式。使用table-cell + vertical-align 通过将父元素转化为一个表格单元格显示 display：table-cell（类似 <td> 和 <th>），再通过设置 vertical-align：middle，使表格单元格内容垂直居中；使用 flex + align-items 。通过设置flex布局中的属性 align-items：center，使子元素垂直居中。

# 居中的盒子不定高

  • 使用 定位 + translateY()。设置“子绝父相”，再设置子元素的 top 为 50% 、 transform 为 translateY(-50%) 即可。

  • 使用table-cell + vertical-align 。通过将父元素转化为一个表格单元格显示 display：table-cell（类似

    和 ），再通过设置 vertical-align：middle，使表格单元格内容垂直居中。

# 定宽定高

利用定位 position + 负margin / translate() 。设置“子绝父相”，绝对定位元素的 top 和 left 均为 50%，再设置其 margin-top 为自身高度的负50%、margin-left 为自身宽度的负50% / transform 的 translate(-50%,-50%) 即可。

# 未知宽高

  • 利用定位 position + translate。设置“子绝父相”，其中绝对定位元素的 top 和 left 都为 50%、transform 的 translate(-50%,-50%)。

  • 利用 position + margin。设置“子绝父相”，子元素设置 top、bottom、left、right 都为0，再设置 margin：auto 即可。

  • 利用 flex。直接设置父元素的display: flex; justify-content: center; align-items: center;即可。
