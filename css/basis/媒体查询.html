# 媒体查询 @media

用于响应式设计。

用法：@media not|only mediatype(all 用于所有多媒体类型设备 print 用于打印机 screen 用于电脑屏幕，平板，智能手机等。speech 用于屏幕阅读器) and (expressions) { ... }

not: not是用来排除掉某些特定的设备的，比如 @media not print（非打印设备）。

only: 用来定某种特别的媒体类型。all则是所有设备。例如:

@media screen and (max-width: 960px){
  body{
        background:#000;
    }
}

# web字体 @font-face

  • 字体格式：.ttf 、.otf 、.woff 、.woff2（推荐） 、.eot（IE） 、.svg

使用方法。

@font-face {
  font-family: name;
  src: url("**") format("woff2"),
       url("**") format("woff");
}
