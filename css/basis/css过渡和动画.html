# 前言

过渡和动画的初衷就是为了使得我们的网页“活”起来，看上去更有灵性。

# 过渡 transition

过渡有以下属性。

  • transition-property 规定应用过渡的 CSS 属性的名称。必填项。

支持的属性有：background-color、background-image（仅支持渐变gradient）、background-position、border-color、border-width、bottom、color、font-size、font-weight、height、left、letter-spacing、line-height、margin、max-height、max-width、min-height、min-width、opacity、outline-color、outline-offset、outline-width、padding、right、text-indent、text-shadow、top、vertical-align、visibility、widtha、word-spacing、z-index、zoom。

  • transition-duration 定义过渡效果花费的时间（以秒s或毫ms秒计）。默认是 0，即没有过渡效果。必须项。

  • transition-timing-function 规定过渡效果的时间曲线。默认是 ease（慢-快-慢）。值可为linear（稳定的速度）、ease-in（慢-快）、ease-out（快-慢）、ease-in-out（慢-快-慢）、cubic-bezier(n,n,n,n)（贝塞尔曲线，n值为0-1之间）。

  • transition-delay 规定过渡效果何时开始（以秒s或毫ms秒计）。默认是 0。

使用的方法：直接指定当前要过渡的css属性和所需过渡花费的时间。即 transition：属性 花费的时间。然后在设置触发的条件（比如鼠标hover，点击等）及属性即可。

# 动画 animation

元素从一种样式逐渐变化为另一种样式的效果,可以看作是过渡的进一步增强。有以下属性：

  • animation-name 规定 @keyframes 动画的名称。

  • animation-duration 规定动画完成一个周期所花费的秒或毫秒。默认是 0。

  • animation-timing-function 规定动画的速度曲线。默认是 "ease"。其他属性值参照过渡效果对应的属性。

  • animation-fill-mode 规定当动画执行完毕后状态。值可为 none（默认值，在动画执行之前和之后不会应用任何样式到目标元素，即以前啥就啥样）、forwards（停留在在动画结束后、backwards（停留在动画开始前）。

  • animation-delay 规定动画何时开始。默认是 0。

  • animation-iteration-count 规定动画被播放的次数。默认是 1。值可为 数值 和 infinite（无数次）

  • animation-direction 规定动画是否在下一周期逆向地播放。默认是 "normal"。值可为reverse（反向播放）、alternate（奇数时正向播、偶数时方向播）、alternate-reverse（正好与alternate相反）

  • animation-play-state 规定动画是否正在运行或暂停。默认是 "running"。

使用方法：先指定动画的 @keyframes 规则，在对元素设置对应的css属性即可。

@keyframes myfirst
{
    0%(相当于关键字from)   {background: red;}
    25%  {background: yellow;}
    50%  {background: blue;}
    100%（相当于关键字to） {background: green;}
}
