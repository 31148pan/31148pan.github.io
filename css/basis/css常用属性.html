# 常用CSS属性

基本属性：width 宽、height 高、padding 内边距、margin 外边距、border 边框。

# background 背景

  • background-color 属性定义了元素的背景颜色，单位值可为：十六进制、rgb(0-255|0-100%,0-255|0-100%,0-255|0-100%)、rgba(0-255|0-100%,0-255|0-100%,0-255|0-100%,0-1)、颜色英文名称（red等）。

  • background-image 属性描述了元素的背景图像.值为 url("引用路径")。

  • background-repeat 属性用于设置元素背景图像水平或者垂直方向是否平铺，值为no-repeat、repeat-x、repeat-y、repeat（默认）、inherit（继承父元素）。

  • background-position 属性用于设置背景图像的起始位置。值可以是left top/center/bottom、center top/center/bottom、right top/center/bottom或者其他CSS单位。

  • background-attachment 背景图像是否固定或者随着页面的其余部分滚动。值为scroll 背景图片随页面的其余部分滚动（默认）、fixed背景图像是固定的。

# text 文本

  • color 用于设置文本颜色，值可为十六进制颜色值、rgb(0-255|0-100%,0-255|0-100%,0-255|0-100%)、rgba(0-255|0-100%,0-255|0-100%,0-255|0-100%,0-1)、颜色名称。

  • text-align 用于设置文本对齐方式 常用属性值left（左对齐）、right（右对齐）、center（居中对齐）、justify（两端对齐）。

  • text-decoration 属性用来设置或删除文本的装饰。值可以为overline（上划线）、underline（下划线）、none（无装饰）、line-through（删除线）。

  • text-indent 指定文本的第一行的缩进。如 text-indent:2em。默认样式下是缩进2个字符

  • letter-space 设置字符之间的间距。属性值为 normal（默认）、数值（px、em、rem）、inherit。

  • word-space 设置单词之间间距。属性值为 letter-space 相同。

  • text-transform 文本转换，是用来指定在一个文本中的大写和小写字母。值可为uppercase（全部大写）、lowercase（全部小写）、capitalize（首字母大写）。

  • text-shadow 设置文本阴影。

水平阴影的位置。值为px、em、rem。允许负值。必需的。

垂直阴影的位置。值为px、em、rem。允许负值。必需的。

模糊的距离。值为px、em、rem。不允许负值。

阴影的颜色。值为CSS颜色。如果不设置该值默认为黑色。

  • vertical-align 属性设置一个元素的垂直对齐方式。

baseline 默认。元素放置在父元素的基线上

top 把元素的顶端与行中最高元素的顶端对齐.

middle 把此元素放置在父元素的中部。

bottom 把元素的底端与行中最低的元素的顶端对齐。

TIP

vertical-align属性起作用的前提必须是作用在内联元素上。 即display值为inline inline-block inline-table table-cell的元素。所以如果元素设置了float: left或者position: absolute，则其vertical-align属性不能生效，因为此时元素的display计算值为block了。

# font 字体

  • font-style 字体样式

italic 斜体。

normal 默认，正常。 ...

  • font-weight 字体粗细

100-900 由细到粗。

bold 粗体。

normal 正常。 ...

  • font-size 字体大小，具体的数值（px、em、rem、%等）。

  • line-height 设置行高。基线到基线之间的距离，以英文字母X为标准。

属性值为 normal 默认。设置合理的行间距。

属性值为 数字，此数字会与当前的font-size相乘来设置行间距。。

属性值为 px 或者 em 等 设置固定的行间距。

属性值为 % 基于当前字体尺寸的百分比行间距。

属性值为 inherit 规定应该从父元素继承。

line-height 设置文本的行高，需要注意的是使用 数值（1、2、3...） 和其它（%、px、em、rem）单位时，继承的特点不一样，数值 是先将数值继承给子元素，在根据子元素字体大小计算，其它的是先根据父元素字体大小计算之后再将结果继承给子元素。

  • font-family 设置文本的字体，可以设置多个，浏览器会根据字体存在的与否，自动从左至右解析。但如果是中文的时候，在设置中一定要给中文加上引号（避免电脑没这种字体出错）。

一般，会采用简写方式：font:***。

# list-style 列表样式

list-style 是list-style-type、list-style-image及list-style-position的缩写。

通常开发中我们会将其设置为list-style:none，这样ul、ol、dl 就不会显示其默认样式。其样式效果会采用其他方式替代，例如：给li标签添加子标签并设置对应样式。

# border 边框。

用于设置元素的边框。

  • border-top/right/bottom/left-width 边框宽度,值为数值。

  • border-top/right/bottom/left-style 边框样式，值为solid（实线）、dashed（虚线）、dotted（点线）、double（双线）等。

  • border-top/right/bottom/left-color 边框颜色，值为颜色值。

::: tips 将元素（也可以是伪元素）的width、height设置为0后，在自定义设置border的TRBL属性可实现自定义三角图形。 :::

# display 元素显示方式

其值可能为inline、block、inline-block、table、table-cell、flex、inline-flex等

display：none 隐藏元素，彻底消失。设置其他属性可再次显示出来。与html标签hidden属性类似。

visibility：hidden 隐藏元素，但该元素还会占据位置。使用visible属性值可以再次显示出来。

TIP

使用html或者css属性显示隐藏元素，DOM结构始终存在。要想从DOM层次控制元素显示隐藏请使用JS方法动态创建、添加或者删除.

# position 元素定位

使用定位属性时一定要配合top、right、bottom及left属性使用，否则可能不会生效。

  • static 静态定位。默认值，即没有定位，遵循正常的文档流对象。

  • absolute 绝对定位。位置相对于最近的已定位父元素，如果元素没有已定位的父元素，那么它的位置相对于html或body。脱离标准流。

  • relative 相对定位。相对定位元素的定位是相对其正常位置。没有脱离标准流。

  • fixed 固定定位。元素的位置相对于浏览器窗口是固定位置。即使窗口是滚动的它也不会移动。脱离标准流。

overflow/overflow-x/overflow-y 设置当元素的内容溢出其区域时的表现。

  • visible 默认值。内容不会被修剪，会呈现在元素框之外。

  • hidden 内容会被修剪，并且其余内容是不可见的。

  • scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。

  • auto auto：根据具体情况判断容器是否要加上滚动条。

# float 浮动

初衷用来设置图片文字环绕效果，元素设置浮动之后，其兄弟元素内的文本会环绕浮动元素的周围。none 默认值（元素不浮动）、left/right 元素向左/右 浮动。

  • 同一个结构下面的浮动找浮动。

  • 浮动以后的元素会覆盖在标准流的元素之上。

  • 浮动的元素只会影响下面的元素，不会影响上面的元素（因为不在同一结构下）。

  • 浮动以后的元素的显示方式与行内块级元素一样,并且脱离了标准流。

浮动的影响：可能会造成父元素高度塌陷，高度不能有子元素撑开。因此需要手动清除浮动。

# clear 清除浮动

  • none，默认值（允许浮动元素出现在两侧）

  • left/right（在 浮动元素 左/右侧不允许浮动元素）。

  • both（两边都不允许出现浮动）。

清除浮动可在浮动元素后面 使用 额外标签法（通常使用伪元素） 或者让其父容器生成 BFC。

# box-shadow 盒子阴影

用于设置盒子的阴影。

  • 水平阴影的位置。值为px、em、rem。允许负值。必填项。

  • 垂直阴影的位置。值为px、em、rem。允许负值。必填项。

  • 阴影模糊的距离 可选项。可以简单理解为阴影的模糊程度，值越大越模糊。值为px、em、rem。不允许负值。

  • 阴影的大小 可选项。

  • 阴影的颜色 可选项。值为CSS颜色。如果不设置该值默认为黑色。

  • inset 可选项,定义阴影的投影方式为内投影，默认值为外投影outset。

特别说明：

当只定义了两个参数，也就是水平和垂直方向的偏移量，默认阴影颜色为盒子字体 color 的属性值。

当定义三个参数时，如 box-shadow:1px 1px 1px 。第三个参数为阴影模糊的距离。

inset 参数只能出现在参数的开头或者结尾。

可以使用逗号 , 创建多重阴影。常用于三边框阴影。例如，box-shadow:2px 2px 5px #333333, 2px 2px 5px #333333。

# background 背景

用于定义盒子的背景。

  • background-image 添加背景图片。不同的背景图像和图像用逗号隔开（background-image：url(""),url("")），所有的图片中显示在最顶端的为第一张。可设置background-position进行定位。

  • background-size 指定背景图像的大小。CSS3以前，背景图像大小由图像的实际大小决定。属性值为长度单位或百分比、contain（自动调整缩放比例，保证图片始终完整显示在背景区域，也就是保证图片的最长端包含在容器内，所以可能会留下空白）、cover（自动调整缩放比例，保证图片始终填充满背景区域，也就是保证图片的最短端包含在容器内，所以部分图片能会溢出，以至于不能完整显示图片）。

  • background-origin 属性指定了背景图像的位置区域(即从什么位置开始放置背景图像，包含该位置)。属性值可为content-box（从内容区开始）、padding-box（默认值，从内边距开始）、border-box（从边框开始）。

  • background-clip 背景剪裁。属性是从指定位置开始绘制。其它位置的背景裁剪掉，注意此时定义的边框始终存在，裁剪的只是背景。属性值可为content-box（从内容区开始，裁剪掉padding和border处背景）、padding-box（从内边距开始）、border-box（默认值，从边框开始）

# 渐变

用于将元素的某种状态渐渐地变成另外的一种状态。

# 线性渐变（Linear Gradients）

  • 渐变的方向，如从上到下（默认情况下）。属性值可以方位（TGBL）也可以是角度（单位是deg，逆时针方向计算，设置角度更容易控制方向）

  • 渐变开始的颜色（必须）

  • 渐变结束的颜色（必须）。可以设置多个颜色结点。如 background-image: linear-gradient(to right, red, green, blue 40%)。

注意：颜色值后面可以设置颜色所占据的百分比。

# 径向渐变（Radial Gradients）

  • 渐变的中心 。默认情况下，渐变的中心是 center（表示在中心点）

  • 形状（圆形或椭圆形）。默认情况下，渐变的形状是 ellipse（表示椭圆形）。值可以是 circle 或 ellipse。如 background: radial-gradient(circle, red, yellow, green)。

  • 大小。默认情况下，渐变的大小是 farthest-corner（表示到最远的角落）。可以是以下四个值：closest-side、farthest-side、closest-corner、farthest-corner

  • 渐变开始/结束的颜色（必须）

# 转换 transform

将元素进行移动、旋转、倾斜。

# 2D转换

  • translate(x,y) 沿着 X 和 Y 轴移动元素。 移动位置相当于自身原来位置，x轴正方向向左，y轴正方向向下。可以为负数。没有脱离标准流。注意设置 % 单位时，是以自身宽高为基准的，如 transform：translate（50%，50%）这里两个 50% 是元素自身宽高的一半。

  • translateX(n) 定义 2D 转换，沿着 X 轴移动元素

  • translateY(n) 定义 2D 转换，沿着 Y 轴移动元素

  • rotate(deg) 可以对元素进行旋转，正值为顺时针，负值为逆时针（当元素旋转以后，坐标轴也跟着发生的转变，可以将旋转操作放到最后）。

  • scale(x, y) 可以对元素进行水平和垂直方向的缩放，x、y的取值可为小数，不可为负值。单位为 数值 或 %( 200% 等同数值 2)。

  • `scaleX(n) 定义 2D 缩放转换，改变元素的宽度(内容可能会被拉伸或者压缩)。

  • scaleY(n) 定义 2D 缩放转换，改变元素的高度。

  • skew(deg, deg) 可以使元素按一定的角度进行倾斜。

可以同时使用多个转换，其格式为：transform: translate() rotate() scale() ...等

# 3D转换

  • translate3d(x,y,z) 。定义 3D 转化。就是比2d多一个z轴而已。translateZ(n) 沿着 Z 轴移动元素。

  • rotate3d(x,y,z)。定义 3D 旋转。

  • rotateX(x) 定义沿 X 轴的 3D 旋转。

  • rotateY(y) 定义沿Y轴的3D旋转。

  • rotateZ(z) 定义沿Z轴的3D旋转。不设置其他3D转换的情况下，与 rotate() 效果相同。

往往设置了2D或3D 转换后，可能还会设置以下属性：

  • perspective 规定 3D 元素的透视效果。值为数值。透视会产生“近大远小”的效果

  • backface-visibility 定义元素在不面对屏幕时是否可见。值可为visible（默认）、hidden。

  • transform-style 规定被嵌套元素如何在 3D 空间中显示。值可为flat（默认表示所有子元素在2D平面呈现）、preserve-3d（转为3d平面显示）;

左手坐标系：伸出左手，让拇指和食指成“L”形，大拇指向右，食指向上，中指指向前方。这样我们就建立了一个左手坐标系，拇指、食指和中指分别代表X、Y、Z轴的正方向。 所有的3D操作都以此为基础进行。
