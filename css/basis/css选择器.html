# 选择器分类

# 基本选择器

# 标签选择器

tagName { key: value }

用于选择标签名是 tagName 的所有标签。

# 类选择器

.className { key: value }

用于选择类名是 className 的所有标签。

# id选择器

#idName { key: value }

将选择 id 属性值是 idName 的一个标签。

# 通配符选择器

*{ key: value }

将选择 body 下的所有标签。

# 关系选择器

# 子代选择器

将某个元素的直接子元素选择出来。

例如：div>.son {color: red } 会找到div下面的类名叫做 son 的直接子元素设置成红色。

# 后代选择器

将某个元素下面的所有的满足条件的后代元素设置相同的样式（子子孙孙）。

例如： div .son{color: red } 凡是在div下面的类名叫做 son 的标签的内容都会显示为红色。

# 交集选择器

会将两个集合中的相同的元素设置样式。

例如：·.className {color:red } 页面上所有的类名叫做 className 的 p 标签内容会显示为红色。

# 并集选择器

会将两个集合中的选择出来的元素都设置相同的样式。

例如：p,div{color: red} 用来将页面上所有的 p 标签和 div 标签的内容都设置为红色 。

# 相邻兄弟选择器

会选择元素E紧邻着的兄弟元素F（EF有相同的父结点，并且F紧跟在E的后面）。

例如：E+F { color:red } 可将紧跟在E元素后面的兄弟元素F的内容设置成红色。

# 通用兄弟选择器

会选择元素E后面的所有兄弟元素F(EF有相同的父节点，F在E之后，不一定紧跟着E)。

例如：E~f{ color:red } 用来匹配设置E元素后面所有的兄弟元素F,并将其文本设置成红色。

# 属性选择器

  • E[attr] 选择所有属性名为 attr 的元素E。

  • E[attr="val"] 选择具有 attr 属性且属性值等于val的E元素。

  • E[attr~="val"] 选择具有 attr 属性且属性值其中一个等于 val 的E元素（包含属性值只有val）。

  • E[attr^="val"] 选择具有 attr 属性且属性值是以val开头的元素E。

  • E[attr|="val"] 选择具有attr属性且属性值是以val开头并以连接符 -分割的元素E（包含属性值只有val的情况）。

  • E[attr$="val"] 选择具有attr属性且属性值是以val结尾的元素E。

  • E[attr*="val"] 选择具有attr属性且属性值包含val的元素E。

# 伪类选择器

  • E:link (超链接未访问时的样式) / E:visited（超链接已访问过的样式） /
    E:hover（鼠标悬停时样式） / E:avtive（设置元素在被用户激活（在鼠标点击与释放之间发生的事件）时的样式）。

  • E:focus设置元素在成为输入焦点（该元素的 onfocus事件发生）时的样式。(一般应用于表单元素)。

  • E:checked 匹配用户界面上处于选中状态的元素E。(用于 input type 为 radio 与 checkbox 时)。

  • E:enabled / E:disabled 匹配用户界面处于可用状态/禁用的元素E（一般用于表单元素）。

  • E:required 用于匹配页面上所有属性为 required的表单元素。

  • E:invalid / E:valid 匹配页面上输入 不合法 / 合法 验证的表单元素。

  • E:empty 匹配没有任何子元素（包括text节点）的元素E。

  • E:not(s) 匹配不含有s选择符的元素E。

  • E:first-child / E:last-child 匹配父元素下的 第一个/最后一个 子元素E。

  • E:only-child 匹配父元素仅有的一个子元素E。

  • E:first-of-type / E:last-of-type 匹配父元素下同类型中的 第一个/最后一个 兄弟元素E 。

  • E:only-of-type 匹配父元素下同类型中的唯一的一个同级兄弟元素E。

  • E:nth-child(n) / E;nth-last-child(n) 匹配父元素的 第n个 / 倒数第几个 子元素E。

  • E:nth-of-type(n) / E:nth-last-of-type(n) 匹配父元素下同类型中的 第n个 / 倒数第n个 同级兄弟元素E。

TIP

E:first-of-type 总是能命中父元素的第1个为 E 的子元素，不论父元素第1个子元素是否为 E；
而 E:first-child 里的 E 元素必须是它的兄弟元素中的第一个元素，否则匹配失效。
>E:last-of-type 与 E:last-child 也是同理。
E:nth-of-type(n) 总是能命中父元素的第n个为 E 的子元素，不论父元素第 n 个子元素是否为 E；而 E:nth-child(n) 会选择父元素的第 n 个子元素 E，如果第 n 个子元素不是 E，则是无效选择符，但 n 会递增。

# 伪元素选择器

  • E:before/E::before 在目标元素E的前面插入的内容。用来和content属性一起使用。

  • E:after/E::after 在目标元素E的后面插入的内容。用来和content属性一起使用。 after 与 before 伪元素都是元素 E 的子元素。

  • E:first-letter / E::first-letter 设置元素内的第一个字符的样式

  • E:first-line / E::first-line 设置元素内的第一行的样式

  • E::placeholder 设置元素文字占位符的样式。(一般用于input输入框)

  • E::selection 设置元素被选择时的字体颜色和背景颜色。

TIP

::placeholder 及其他伪元素在使用时需要加上各个浏览器的前缀（-moz- 、-webkit-、-o-、-ms-）。一般会使用插件autoprefix处理。
