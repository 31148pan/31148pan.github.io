# 样式自定义

大部分html标签在浏览器显示的时候都会有一个默认的属性值：如：

  • body、p 和 h标签默认的margin。

  • 列表元素ul、ol 和 dl 都有默认的margin和padding。

  • 。。。

往往我们都会引入reset.css重置该标签的默认样式。然后自定义我们所需的样式

下面是滑块自定义样式范例。

# 自定义滑块样式

滑块（range）是表单控件中常见的一种，常用于音频控制。所以学会使用和自定义它的样式很是必要。注意IE9以下不支持该控件。

# 如何使用滑动条？

用法很简单，如下所示

 <input type="range" value="0">
1

该控件常用有以下几个html属性：

autofocus 设置或返回滑块控件在页面加载后是否应自动获取焦点。
defaultValue 设置或返回滑块控件的默认值。
disabled 设置或返回滑块控件是否被禁用。
max 设置或返回滑块控件的 max 属性值，默认为100。
min 设置或返回滑块控件的 min 属性值，默认为0。
step 设置或返回滑块控件的 step 属性值，默认为1。
value 设置或返回滑块控件的 value 属性值，默认为0。

# 如何美化滑动条？

方案一：通过css改造样式。 方案二：通过将滑动条元素的透明度设置为0（opacity:0），在自定义div实现（失去该控件的存在意义，这里为了方便不采用）

方案一实现如下：

input[type="range"] {
    /* 去除浏览器默认样式 */
    outline: none;
    -webkit-appearance: none;
    /* 自定义样式 */
    width: 200px;
    border-radius: 10px;
    background-image: linear-gradient(#059CFA, #059CFA);
    background-repeat: no-repeat;
    background-size: 0% 100%;
  }

  /* 自定义滑块样式 */

  /* chrome */
  input[type=range]::-webkit-slider-thumb {
    -webkit-appearance: none;
    height: 15px;
    width: 15px;
    /*使滑块超出轨道部分的偏移量相等*/
    background-color: pink;
    border-radius: 50%;
    /*外观设置为圆形*/
    border: solid 1px rgba(205, 224, 230, 0.5);
    /*设置边框*/
    box-shadow: 0 1px 1px #3b4547;
    /*添加底部阴影*/
  }

  /* firefox */
  input [type=range] ::-moz-range-thumb {
    -webkit-appearance: none;
    height: 15px;
    width: 15px;
    /*使滑块超出轨道部分的偏移量相等*/
    background: pink;
    border-radius: 50%;
    /*外观设置为圆形*/
    border: solid 1px rgba(205, 224, 230, 0.5);
    /*设置边框*/
    box-shadow: 0 1px 1px #3b4547;
    /*添加底部阴影*/
  }

  /* 自定义滑块轨道样式 */
  /* chrome */
  input[type=range]::-webkit-slider-runnable-track {
    height: 15px;
    /*将轨道设为圆角的*/
    border-radius: 10px;
    /*轨道内置阴影效果*/
    box-shadow: 0 1px 1px #998, inset 0 1px 1px #0d1112;
  }

  /* firefox */
  input[type=range]::-moz-range-track {
    height: 15px;
    /*将轨道设为圆角的*/
    border-radius: 10px;
    /*轨道内置阴影效果*/
    box-shadow: 0 1px 1px #998, inset 0 1px 1px #0d1112;
  }

  /* firefox 下不需要js填充进度条，直接添加下面代码即可 */
  input[type=range]::-moz-range-progress {
    background: #059CFA;
    height: 13px;
    border-radius: 10px;
  }

  /* IE */
  input[type=range]::-ms-track {
    height: 25px;
    border-radius: 10px;
    box-shadow: 0 1px 1px #def3f8, inset 0 1px 1px #0d1112;
    border-color: transparent;
    /*去除原有边框*/
    color: transparent;
    /*去除轨道内的竖线*/
  }

  input[type=range]::-ms-thumb {
    border: solid 1px rgba(205, 224, 230, 0.5);
    height: 25px;
    width: 25px;
    border-radius: 50%;
    background: pink;
    box-shadow: 0 1px 1px #3b4547;
  }

  input[type=range]::-ms-fill-lower {
    /*进度条已填充的部分*/
    height: 20px;
    border-radius: 10px;
    background: #059CFA;
  }

  input[type=range]::-ms-fill-upper {
    /*进度条未填充的部分*/
    height: 20px;
    border-radius: 10px;
    background: #ffffff;
  }
