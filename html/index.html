# 简介

# HTML定义

  • HTML（英文缩写 HyperText Markup Language），简言之，它就是一种超文本标记语言。是一种用来结构化 Web 网页及其内容的标记语言。主要功能是用来描述页面。
  • 由 尖括号 包裹的关键词标签，比如 <html> ,该标签表示当前文档中的根元素。

# 特点

  • 标签的书写规则不区分大小写，但是推荐使用小写 ：在未来(X)HTML 版本中可能会强制使用小写。
  • 通常以开始标签 开始，结束标签 结束，标签要尽量正确的关闭，否则可能会出现意想不到的错误。
  • 单标签（又称自闭合标签）在HTML5 中可以不用正确的关闭，比如 换行<br> 这是推荐的做法，但在 XML 和 HTML4.0 必须正确关闭自己，比如 <br/> 和 <p> content </p>。

# 基本结构

<!--文档类型声明，用于告知浏览器当前页面使用了哪种 HTML 版本，防止浏览器怪异解析。-->
<!DOCTYPE html>
<!--该标签包含了整个网页的所有内容，有时也被称作根元素 -->
<html>
<!--该标签规定了文档相关的配置信息（元数据），包括文档的字符编码、标题，引用的文档样式和脚本等。-->
<head>
<!--网页标题-->
<title>document</title>
</head>
<!--该标签用于定义文档的主体，其内部包含了所有用户可以看到的内容，可以是文本、图像、视频、音频等内容。-->
<body>
   some code...
</body>

</html>
