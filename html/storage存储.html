# Web 存储（storage）

# localStorage 本地存储

用于长久保存整个网站的数据，保存的数据没有过期时间，直到手动去除。只能存储字符串，可以将对象JSON.stringify() 编码后存储。解码使用JSON.parse()。

# sessionStorage 会话存储

用于临时保存同一窗口(或标签页)的数据，在关闭窗口或标签页之后将会删除这些数据。

TIP

不管是 localStorage，还是 sessionStorage，可使用的API都相同。如果想设置数据的生命周期，可以在存入数据时加上一个时间戳，动态计算时间是否过期，在进行删除数据操作即可。

常用的有如下几个（以localStorage为例）:

  • 保存数据：localStorage.setItem(key,value);
  • 读取数据：localStorage.getItem(key);
  • 删除单个数据：localStorage.removeItem(key);
  • 删除所有数据：localStorage.clear();
  • 得到某个索引的key：localStorage.key(index);
  • localStorage.length ：查看所存储的数据长度。

使用前应该检测浏览器是否支持该方法


 if(Storage){
    // 支持 localStorage  或者sessionStorage 对象!
    // 一些代码.....
} else {
    // 抱歉! 不支持 web 存储。
}
